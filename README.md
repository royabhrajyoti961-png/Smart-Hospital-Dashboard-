# Smart-Hospital-Dashboard-
Smart Hospital Dashboard is an AI-powered system built with Streamlit that monitors patient data in real time. It tracks vitals, predicts risk levels, and alerts staff instantly. The platform improves efficiency, enables faster decisions, and enhances patient care through automation and analytics.

# Smart Hospital Dashboard

## Features
- Patient monitoring
- AI risk prediction
- Real-time dashboard
- Streamlit UI
